# Quant for CN_A

[English](./README.md)

这是一个面向个人投资者的 A 股可视化优先量化分析项目。

## 当前阶段

项目目前处于规划与 MVP 定义阶段。

第一阶段目标不是立刻搭建本地网页，而是先完成：

1. 指定指数和个股的公开数据采集
2. 指数长期趋势分析
3. 个股长期趋势分析
4. 可重复生成的可视化结果与分析输出

等数据链路和可视化稳定后，再搭建本地 `localhost` 网站，并在后续部署到 Azure。

## 当前 MVP 方向

当前 MVP 重点是指数和个股分析，而不是短线预测：

- 只做 A 股
- 重点关注 `上证指数` 和 `沪深300`
- 围绕少量代表性个股做长期趋势分析
- 输出可解释的趋势判断，而不是几天内的涨跌预测
- 板块和行业分析推迟到后续阶段

## 数据源方案

当前主数据源选择为 `efinance`。

选择原因：

- 对 Python 友好，适合学生项目快速起步
- 支持通过 `ef.stock.get_quote_history()` 获取股票历史行情
- 支持通过 `ef.stock.get_base_info()` 获取个股基础信息
- 支持通过 `ef.stock.get_realtime_quotes()` 获取指数行情快照
- README 中明确提到遇到限流或网络报错时可以尝试 `TickFlow`

当前备用方案：

- 当 `efinance` 不稳定、限流或接口异常时，使用 `TickFlow` 作为备用数据源

## 推荐的起步顺序

1. 明确两个指数和候选个股需要的字段
2. 验证 `efinance` 对指数历史行情、个股历史行情、个股基础信息的覆盖情况
3. 建立本地数据库与更新脚本
4. 实现长期趋势分析指标
5. 生成 notebook 或脚本式可视化输出
6. 后续再接本地网页
